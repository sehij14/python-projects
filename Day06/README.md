Day 6 - Student Record Manager using Lists

About the Project:

This project is based on Python Lists.
In this program, I created a simple Student Record Manager that stores and manages student names using lists.

The project helped me understand how multiple values can be stored together and managed dynamically in Python.

---

Features:

- Add student names
- View student records
- Store multiple student entries
- Manage records using lists
- Simple menu-based interaction

---

Concepts I Used:

- Python Lists
- "append()"
- loops
- user input
- indexing
- basic menu system
- conditional statements

---

File Included:

What I Learned:
