Day 11 - Student Class System

About the Project:

This project is based on Object-Oriented Programming (OOP) in Python.

In this program, I created a Student Class System to understand how classes and objects work in Python. Instead of writing programs using only variables and functions, this project introduced a more organized way of designing programs using objects.

The system stores student-related data inside a class and performs operations using object properties and methods.

This was my first proper step into OOP concepts, which are widely used in real software development.

---

Features:

- Create student objects
- Store student details
- Display student information
- Use methods inside classes
- Access object attributes

---

Concepts Used:

- classes and objects
- constructors
- "self"
- methods
- object creation
- instance variables
- basic OOP structure
