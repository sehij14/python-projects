Day 11 - Student Class System

About the Project:

This project is based on Object-Oriented Programming (OOP) in Python.

In this program, I created a Student Class System to understand how classes and objects 
work in Python. Instead of writing programs using only variables and functions, this 
project introduced a more organized way of designing programs using objects.

The system stores student-related data inside a class and performs operations using 
object properties and methods.

This was my first proper step into OOP concepts, which are widely used in real software 
development.

---

Features:

- Create student objects
- Store student details
- Display student information
- Use methods inside classes
- Access object attributes

---

Concepts Used:

- classes and objects
- constructors
- "self"
- methods
- object creation
- instance variables
- basic OOP structure

---

File Included:

day11_student_class_system.py

---

What I Learned:

- Difference between procedural programming and OOP.
- How classes help organize data and behavior together.
- How objects are created from classes.
- Why OOP is important in larger applications.

This project helped me understand the foundation of Object-Oriented Programming in 
Python.

---

Why I Made This Project:

The purpose of this project i made was to move from basic programming concepts toward 
structured programming using Object-Oriented Programming.

Most real-world applications are designed using OOP concepts because they help manage 
larger systems more efficiently.

---

How to Run:

1. Open the Day 11 project folder.
2. Open "student_class_system.py"
3. Run the program.
4. The program will create and display student object details.
