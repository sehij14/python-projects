# Smart File Organizer Utility - Day 34

This project is a small productivity utility built using Python and Object Oriented Programming concepts.

The idea behind this project came from how real systems categorize and process files automatically. Instead of building another basic CRUD-style project, I wanted to create something that felt more practical and utility-oriented.

One of the key highlights of this project was implementing a utility-based architecture. 

---

The program accepts filenames from the user and categorizes them into groups such as:
- Images
- Documents
- Videos
- Audio
- Others

The project also validates filenames before processing them.

---

One thing I liked while building this project was understanding how utility classes are used in real applications. In many production systems, helper functions are grouped together for cleaner architecture instead of being written randomly throughout the codebase.

---

## Concepts practiced in this project:
- *Object-Oriented Programming*: Class organisation and strategic use of *Static methods*
- *Data Handling*: Advanced usage of *Lists*, *Loops*, and *String methods*.
- *Validation logic*
- *Menu-driven systems*
