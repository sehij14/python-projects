Day 7 - Student Database System using Dictionaries

About the Project:

This project is based on Python Dictionaries.

In this program, I created a simple Student Database System that stores and manages student information using dictionaries.

The project helped me understand how data can be stored in key-value format and accessed efficiently in Python.

---

Features:

- Store student names and marks
- View student records
- Search student details
- Manage multiple student entries
- Menu-based system

---

Concepts Used:

- Python Dictionaries
- key-value pairs
- loops
- user input
- conditional statements
- menu-driven programming

---
