Day 9 - Safe Calculator System with Error Handling

About the Project:

This project is based on Error Handling in Python.

In this program, I created a Safe Calculator System that performs basic mathematical operations while handling the possible user input errors properly.

Earlier calculator programs could crash if the user entered invalid values or performed operations like division by zero. In this project, I improved the program by adding error handling so the system can manage such situations safely.

This project helped me understand why real-world applications need proper validation and protection against unexpected errors.

- Addition

- Multiplication
- Division
- Handles invalid input safely
