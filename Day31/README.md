#Training Institute Management System
This project is part of my Python and Software Engineering learning journey.

This is a simple Python project based on Object Oriented Programming concepts.
The program allows users to add student details and display all stored students using a menu-driven system.

I built this project to practice:
- Classes and Objects
- Constructors
- Methods
- Lists
- Error Handling
- Menu-driven programs

Features of this project:
- Add student details
- Display all students
- Input validation using try-except
- Object-oriented structure

Technologies I Used:
- Python
- VS Code
- GitHub and Git

How to Run:
1. Open the project folder.
2. Run the file:
   python training_institute_management_system.py

What I Learned:
- How classes and objects work.
- How constructors initialize object data.
- How methods organize functionality.
- Why OOP is important in software development.
- This project helped me think more like a software engineer instead of only writing small scripts.
