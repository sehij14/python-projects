Day 5 - Student Report System-Marks Checker and Result Analysis

About the Project:

This project is based on conditional statements in Python.

In this program, I created a simple marks checker that analyzes the students marks and displays the results based on different conditions.

The program checks:

- pass or fail
- grade conditions
- different outputs based on marks entered by the user

This project helped me understand how programs make decisions using conditions.

---

Features:

- Takes marks input from the user
- Checks pass or fail status
- Displays different results using conditions
- Uses logical decision-making

---

Concepts Used:

- "if"
- "elif"
- "else"
- comparison operators
- user input
- integer conversion
- conditional logic

---

File Included:

What I Learned:
